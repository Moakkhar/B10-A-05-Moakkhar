What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each object corresponds to a part of the document. Through the DOM, programs (like JavaScript) can dynamically access, modify, add, or delete elements from a webpage, essentially allowing web developers to create interactive, dynamic web experiences.

How do you select an element from the DOM?

To select an element from the DOM, use methods like:
document.getElementById('id') Selects an element by its id.
document.querySelector('selector') Selects the first element matching a CSS selector.
document.querySelectorAll('selector') Selects all elements matching a CSS selector.

What is event delegation in the context of the DOM, and Why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events for its child elements, instead of attaching multiple listeners to each child. Why it's useful:
1. Efficiency: Reduces memory usage by minimizing the number of event listeners.
2. Dynamic Handling: Easily manages events for dynamically added elements.
3. Simplifies Code: Streamlines event handling for large numbers of child elements.
It leverages event bubbling, where events triggered on child elements propagate up to their parent, making this technique possible.

How do you manipulate an element's attributes and style using the DOM?

To manipulate an element's attributes and style using the DOM:
1. Attributes:
Use setAttribute('attr', 'value') to set an attribute. Example: element.setAttribute('src', 'image.jpg'). Use getAttribute('attr') to get an attribute's value. Use removeAttribute('attr') to remove an attribute.
2. Style:
Access the style property to change inline CSS. Example: element.style.color = 'red'; to change the text color. Use classList.add(), classList.remove(), or classList.toggle() to manipulate CSS classes.
